#### 后端技术

技术 | 状态 | 说明 |
----|----|----
Spring Boot | 容器+MVC框架 |
Spring Security | 认证和授权框架 |
MyBatis | ORM框架  |
MyBatisGenerator | 数据层代码生成 |
PageHelper | MyBatis物理分页插件 |
MySQL |MySQL数据库 |
Druid | 数据库连接池 |
Swagger-UI | 文档生产工具 |
JWT | JWT登录支持 |
Log4j  | 日志 |
Springboot devtool | Springboot热部署等 |
